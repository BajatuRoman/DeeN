How to encrypt:
1.Open "IN.txt" and write your encrypted message
2.Run "Deen.exe"
3.Find your decrypted message in "OUT.txt"

How to decrypt:
1.Open "IN.txt" and write your message
2.Run "Deen.exe"
3.Find your encrypted message in "OUT.txt"

If you get "ERROR":
1.Your message must contain ONLY letters, numbers and spaces
2.Your message must be less than 90 characters long
